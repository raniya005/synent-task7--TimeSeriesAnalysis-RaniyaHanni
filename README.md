# synent-task7--TimeSeriesAnalysis-RaniyaHanni
Problem Statement:
The objective of this project is to analyze stock market trends over time and forecast future stock prices using Time Series Analysis and Linear Regression techniques.
Dataset Overview:
Dataset: Stock Price Dataset
The dataset includes:
- Date
- Stock value
- Turnover
- Market-related attributes
The dataset was used to analyze trends, yearly averages, and future price predictions.
Approach:
1. Loaded and cleaned the stock market dataset
2. Converted date columns into datetime format
3. Performed trend analysis using line plots
4. Built a Linear Regression model for future forecasting
5. Visualized predicted future stock prices
Result:
- Stock prices fluctuate significantly over time
- Yearly analysis indicates overall market growth
- Future forecasting predicts potential upward price movement
- Trading activity varies across different periods
